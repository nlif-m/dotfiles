# Dependency

```sh
gnu stow
gnu make
```

## Install dependencies

### Arch/Artix linux
```sh
sudo pacman -S stow make
```
### Debian/Ubuntu/Linux Mint
```sh
sudo apt install stow make
```
# Install guide

```sh
git clone https://github.com/nlif-m/dotfiles ~/.dotfiles
cd ~/.dotfiles
make install
```
